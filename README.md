# OverlordOrder
Overlord Order

## Software
* Java 21
* Micronaut
* MariaDB
* Gradle 8.11.1

## Format
```
gradle spotlessApply
```

## Date Format
```
@JsonFormat(pattern = "yyyy-MM-dd'T'HH:mm")
```

### Run
```
gradle run 
```
### Run Hot Reload
```
gradle run --continuous
```
### Task
```
gradle task
```
### Compile
```
gradle assemble
```
or
```
gradle build
```
